# Jazzian

## Instructions

```
git clone https://github.com/Gonzalesderkaefer/Jazzian
cd Jazzian
./install.sh
```

## Supported Distros
- Debian-based (only tested on debian tough)
    - awesomewm
    - i3
    - bspwm
    - sway
- Fedora
    - awesomewm
    - i3
    - bspwm
    - sway
    - river
    - hyprland
- Arch Linux
    - awesomewm
    - i3
    - bspwm
    - sway
    - river
    - hyprland
